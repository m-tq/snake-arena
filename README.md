# Snake Arena

Realtime multiplayer snake game built with Node.js + WebSocket.

## Features
- Realtime multiplayer with rooms & spectators
- Game modes: Last Standing, Timed, Free Play
- Snake color & pattern customization
- Leaderboard & player stats

## Prerequisites
- Node.js 16+
- npm

## Install & Run

```bash
npm install
npm start
```

Open `http://localhost:3000`.

## Scripts
- `npm start` runs the server
- `npm run dev` runs the server with watch

## Project Structure
```
server/   # Backend + WebSocket
public/   # Frontend (HTML/CSS/JS)
data/     # SQLite database
```
